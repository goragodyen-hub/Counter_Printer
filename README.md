# 🖨️ Counter Printer Tracker — Cloud & Static Web App

ระบบบันทึกและคำนวณมิเตอร์เครื่องพิมพ์รายเดือน สำหรับใช้ร่วมกัน 2 คนแบบไร้สาย (Serverless) ทำงานโดยการเก็บบันทึกข้อมูลตรงไปยัง **Google Sheets** ผ่าน **Google Apps Script** และสามารถนำขึ้นโฮสต์ฟรีบน **GitHub Pages** เพื่อเปิดใช้งานจากโทรศัพท์มือถือหรือคอมพิวเตอร์ที่ใดก็ได้โดยไม่ต้องเปิดเครื่องคอมพิวเตอร์ localhost ทิ้งไว้!

---

## ⚡ สถาปัตยกรรมใหม่ (Serverless Cloud)

```
📱 เบราว์เซอร์ (GitHub Pages / Local)  ───(HTTPS/JSON)───>  ☁️ Google Apps Script (Web App)  ───>  📊 Google Sheets (ฐานข้อมูลบนคลาวด์)
```

### สิ่งที่เปลี่ยนแปลง:
- **ไร้ระบบ Express/Node.js**: ลบ `server.js`, `package.json`, และ `node_modules` ออกทั้งหมด ทำให้โค้ดมีน้ำหนักเบาและดูแลรักษาง่าย
- **คลาวด์ 100%**: ข้อมูลทั้งหมดปลอดภัยบนระบบ Google Sheets ของคุณเอง
- **ปลอดภัย & พร้อมกัน**: ทุกคนสามารถแก้ไข เพิ่มเติม หรือลบข้อมูลเครื่องพิมพ์และบันทึกมิเตอร์ได้จากทุกอุปกรณ์พร้อมกัน

---

## 🛠️ ขั้นตอนการตั้งค่าระบบ (ทำครั้งแรกครั้งเดียว)

### 1. เตรียม Google Sheet
1. เปิด [Google Sheets](https://sheets.google.com/) และสร้างสเปรดชีตใหม่ (เช่น ตั้งชื่อว่า `Counter Printer Data`)
2. ในสเปรดชีต ให้สร้างแท็บ Sheet ขึ้นมา **2 แท็บ** และพิมพ์ชื่อแถวแรก (Header) ตามนี้:
   - แท็บ **`Printers`** (ช่อง A1 ถึง G1):
     | id | zone | location | ip | serial | type | note |
   - แท็บ **`Records`** (ช่อง A1 ถึง G1):
     | id | printerId | month | counterBW | counterColor | recordedAt | note |

### 2. ติดตั้ง Google Apps Script (Backend API)
1. ในหน้า Google Sheet ของคุณ ไปที่เมนูด้านบน: **ส่วนขยาย (Extensions)** → **แอปส์สคริปต์ (Apps Script)**
2. ลบโค้ดเริ่มต้นในไฟล์ `รหัส.gs` (หรือ `Code.gs`) ออกให้หมด
3. เปิดไฟล์ [google-apps-script.txt](file:///d:/My%20project/Counter%20Printer/google-apps-script.txt) ในโปรเจกต์นี้ คัดลอกโค้ดทั้งหมดแล้วนำไปวางในช่องเขียนโค้ดของ Apps Script
4. กดปุ่ม **บันทึก (Save)** (รูปแผ่นดิสก์)

### 3. Deploy เป็น Web App
1. กดปุ่ม **การทำให้ใช้งานได้ (Deploy)** (สีน้ำเงินขวาบน) → เลือก **การทำให้ใช้งานได้ใหม่ (New deployment)**
2. คลิกรูปเฟืองข้างคำว่า Select type เลือก **เว็บแอป (Web app)**
3. ตั้งค่าข้อมูลดังนี้:
   - **คำอธิบาย (Description)**: `Printer Counter API`
   - **เรียกใช้งานในฐานะ (Execute as)**: `ฉัน (Me - อีเมลของคุณ)`
   - **ผู้มีสิทธิ์เข้าถึง (Who has access)**: `ทุกคน (Anyone)` *(สำคัญมาก: หากตั้งค่าเป็นคนอื่น เบราว์เซอร์จะไม่สามารถเขียนข้อมูลได้)*
4. กดปุ่ม **การทำให้ใช้งานได้ (Deploy)**
5. ระบบอาจขอสิทธิ์เข้าถึงบัญชี Google ให้กด **ให้สิทธิ์เข้าถึง (Authorize access)** → เลือกบัญชีของคุณ → กด **Advanced** → กด **Go to ... (unsafe)** → กด **Allow**
6. เมื่อ Deploy สำเร็จ ให้คัดลอก **URL เว็บแอป (Web app URL)** ที่ได้เก็บไว้

### 4. เชื่อมโยงหน้าเว็บเข้ากับ Google Sheets
1. เปิดไฟล์ [app.js](file:///d:/My%20project/Counter%20Printer/app.js) ในโปรเจกต์นี้
2. หาบรรทัดที่ `let gsheetsUrl = '...';` (ประมาณบรรทัดที่ 13)
3. วาง URL เว็บแอปที่คุณคัดลอกมาแทนที่ URL เดิมในเครื่องหมายคำพูดเดี่ยว
4. บันทึกไฟล์ `app.js`

## 🚀 วิธีนำเข้าข้อมูลเดิมจากเครื่อง (Migration)

เนื่องจากระบบเก่าบันทึกข้อมูลไว้ในไฟล์ `data/printers.json` และ `data/records.json` คุณสามารถโอนย้ายข้อมูลทั้งหมดขึ้น Google Sheets ได้ในคลิกเดียว:

1. เปิดรันโปรเจกต์แบบ Local บนเครื่องคอมพิวเตอร์ของคุณก่อน (เช่น ใช้ Extension **Live Server** ใน VS Code หรือเปิดเซิร์ฟเวอร์แบบ static)
2. ไปที่เมนู **ตั้งค่าเครื่องพิมพ์ (⚙️)** บนหน้าเว็บ
3. คลิกปุ่ม **`🚀 นำเข้าข้อมูลเดิม`** (Import) ด้านบนขวา
4. กดตกลงในกล่องข้อความยืนยัน ระบบจะอ่านข้อมูลจากไฟล์ JSON และส่งขึ้นไปบันทึกลงใน Google Sheet ของคุณโดยอัตโนมัติ
5. เปิดตรวจสอบใน Google Sheet และ Dashboard บนหน้าเว็บเพื่อยืนยันความถูกต้อง

## 🌐 วิธีนำขึ้นใช้งานจริงผ่าน GitHub Pages (โฮสต์ฟรี!)

เมื่อโอนข้อมูลขึ้น Google Sheets แล้ว คุณสามารถนำโค้ดขึ้นเว็บออนไลน์ได้ฟรี เพื่อให้ใช้งานร่วมกับผู้ใช้อีกคนได้แบบไร้รอยต่อ:

1. ล็อกอินเข้าใช้งานบัญชี **GitHub** ของคุณ
2. สร้าง **Repository ใหม่** (New Repository) ตั้งชื่อตามต้องการ เช่น `counter-printer`
3. อัปโหลดไฟล์ในโฟลเดอร์นี้ขึ้นไป (ประกอบไปด้วย: `index.html`, `app.js`, `style.css`, `sw.js`, `manifest.json`, และรูปภาพไอคอนต่างๆ)
4. ไปที่แท็บ **Settings** ของ Repository นั้น
5. เลื่อนไปที่เมนู **Pages** ด้านซ้ายมือ
6. ที่หัวข้อ **Build and deployment** → ให้เลือก Branch เป็น **`main`** (หรือ `master`) และโฟลเดอร์เป็น **`/ (root)`**
7. กด **Save**
8. รอประมาณ 1-2 นาที GitHub จะแสดงลิงก์เว็บไซต์ออนไลน์ของคุณ (เช่น `https://<ชื่อผู้ใช้>.github.io/counter-printer/`)
9. คัดลอกลิงก์นั้นไปส่งให้ผู้ร่วมงานใช้งาน หรือเก็บบันทึกเป็นหน้าโฮมสกรีนบนมือถือ (PWA) เพื่อใช้บันทึกได้ทุกที่ทุกเวลา!

## 📁 โครงสร้างไฟล์ในปัจจุบัน

```
Counter Printer/
├── index.html              # โครงสร้างหน้าเว็บ และ UI แดชบอร์ด
├── app.js                  # Logic การเชื่อมต่อ Google Sheets API, การคำนวณ และการแสดงผลกราฟ
├── style.css               # สไตล์สีธีม Dark Mode, micro-animations และหน้าตาปุ่ม/ตาราง
├── sw.js                   # Service Worker สำหรับเก็บ Cache ให้แอปโหลดได้เร็วแม้เน็ตช้า (และข้าม Cache สำหรับ Google Sheets)
├── manifest.json           # โครงสร้างสำหรับการติดตั้งเป็น App บนหน้าจอมือถือ (PWA)
├── icon-192.png / icon-512.png   # รูปไอคอนของระบบ
├── google-apps-script.txt  # โค้ดต้นฉบับที่จะนำไปแปะใน Google Apps Script (ห้ามลบ)
├── data/
│   ├── printers.json       # ข้อมูลเครื่องพิมพ์เดิม (มีไว้สำรองสำหรับนำเข้า)
│   └── records.json        # ข้อมูลมิเตอร์เดิม (มีไว้สำรองสำหรับนำเข้า)
├── .gitignore              # กำหนดไฟล์ที่ไม่ต้องส่งขึ้น Git
└── README.md               # คู่มือการใช้งานเล่มนี้
```
